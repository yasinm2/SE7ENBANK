This application was made during my internship at Intertech. I only had 4 days to make the application, so it was done without following any rules (MVVM, MVC, clean code, etc.). There is only the iOS application in this repo. The actual functions that can recognize the identity were integrated on Flask and then dockerized and deployed to Azure. Unfortunately, I cannot share the codes we wrote in the artificial intelligence section. But I can share this IOS application. Here, I wanted to show you how to connect to web services, how to print error messages on your screen, or what action to take based on the returns coming from the server.
